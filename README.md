# Api Integration Widget

Widget for fast integration.

## npm command

- `npm run dev` - develop with watch and webpack-dev-server
- `npm run build` - build js and css to docs/ folder

## Stack

- Vue.js
- Vue-Router
- Vuex
- ES6/ES7
- Webpack

# Features 
- l2f
- s2s
- open in editor
- work with templates
